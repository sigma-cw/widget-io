How to use the extension

Log in to StreamElements and go to your Overlays Page

Create a new overlay

A new tab will open. Input your desired resolution, and remember it for later!

The widget.io button

If the extension is working properly, a new button should appear right next to the "+" button in the bottom left corner of the editor. Click it!

Upload your widget's .zip file and click "Create Widget"

Note: The extension will automatically import widgets which were exported with it. If the widget you're importing was exported in a different way, the extension will prompt you to point it to each of the HTML, CSS, etc. files

Resize the widget window

When the editor window disappears, you should see a faint blue outline on the visual ediot. Drag it's corners to resize it.

Copy link

Give your overaly a name in the top left, and then click the link icon right next to the preview button in the top right. This will copy the overlay link. Then click save.

Add to OBS

Add a new browser source, and paste the link you just copied into the URL field. Then change the width and height values to match the resolution you set in StreamElements. Click OK and you're done!
